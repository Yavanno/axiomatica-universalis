## Kern

Vier ontologische Axiome (Existenz, Asymmetrie, Irreversibilität, Unbeschränktheit) erzwingen einen eindeutigen spektralen Fixpunkt:

```
β = D / (D + 1)
```

D = effektive Tiefe des Empfängers. Für D → ∞: β → 1 (rosa Rauschen). Bedeutung entsteht im offenen Intervall C ∈ (0,1) — *die Lücke*.
