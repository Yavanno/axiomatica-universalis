## Was ist die Axiomatica Universalis?

Eine formale Bedeutungstheorie auf vier ontologischen Axiomen:

| Axiom | Name | Aussage |
|-------|------|---------|
| A1 | Existenz | Es gibt unterscheidbare Zustände |
| A2 | Asymmetrie | Nicht alle Zustände sind gleich wahrscheinlich |
| A3 | Irreversibilität | Zustände ändern sich in eine Richtung |
| A4 | Unbegrenztheit | Kein System hat eine letzte innere Grenze |

**Kernresultat:** Das 1/f-Spektrum (β = D/(D+1)) ist der einzige skaleninvariante Bedeutungsträger. Jedes System das die vier Axiome erfüllt — Neuron, Stern, Sprachmodell — operiert in der Lücke C∈(0,1).
